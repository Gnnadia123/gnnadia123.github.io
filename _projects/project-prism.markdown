# A new way to play text-based games

Project: Prism was started in early 2026 as a side project to explore areas of python formatting and real-time inputs. Eventually, this testbed grew to a game with dynamic weapon combat, combo chaining, synergy with elements, and an interconnected and interweaved theorethical universe. What started as a simple "click to roll" game inspired by RNG has now let skill-based quicktime events take the stage, possibly the first of its kind for a text-based terminal python game. 

Creating this game was not easy. Currently, the game is still in development, with the Minimum Viable Product (MVP) having gaps and errors every few clicks. However, I will continue to work on this project, for the forseeable future. Having everything click into place at the end makes everything worth it. 

Try the game here [on my github!](https://github.com/Gnnadia123/Project-Prism)

## Features: 
- Quicktime combat
- Combos and synergy
- Elemental synergies 
- Customisation
- Myriad of weapon possibilities 
- RNG rolling for better Prisms
