My latest project, Project: Prism, is a text-based python game about rolling Prisms and using them to fight enemies. Currently, there are just a handful of prisms, but that is set to increase very soon. The quicktime combat (QTC) uses the curses library to track real-time inputs while other UI design elements make use of the Rich library. 

Project: Prism is my most ambitious project as of now, and developing this game was fun and eye-opening as I learnt about the different libraries and mechanics. There is also build in cutscenes, but instead of coding them by hand (through previous experience), I made use of a coordinate system and frame tracking with 2-axes to achieve a more efficient and scalable design. 

The game is in Gen 0-1 (Means version 0.1) and is now up on my github at: https://github.com/Gnnadia123/Project-Prism/tree/main 

This game was inspired by games like Sol's RNG, and offers a text-based break away from every other gacha game nowadays with flashy lights and overloaded effects.
